# Coursera Scraper

> [!IMPORTANT]
> Nothing to see here

## Field to scrape

**Data to scrape manually**
- What you'll learn
- Skills you'll gain
- Difficulty Level
- Outcomes
- Time to complete


**Data I can get from `coursera.org/api/courses.v1/?q=slug&slug={SLUG}`&fields=**
- description
- name
- partnerIds e.g https://www.coursera.org/api/partners.v1?ids=1734
- instructorIds e.g https://www.coursera.org/api/instructors.v1?ids=158375127 

**Data I can get from https://www.coursera.org/api/onDemandCourseMaterials.v2/?q=slug&slug={SLUG}&includes=modules**
- course name
- course description
- time commitment
